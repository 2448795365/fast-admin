技术选型

    1、后端技术
        - Spring Boot       核心框架
        - Apache Shiro      权限框架
        - FreeMarker        模版引擎
        - MyBatis           持久层框架
        - Mybatis-Plus      MyBatis增强工具
        - Druid             数据库连接池
        - MySql             数据库
        - Redis             缓存数据库
        
    2、前端技术
        - layui             前端框架
        - jQuery            JS框架
        - zTree             树插件
        - Font-awesome      字体图标
使用方法

    1、导入sql,目录:resources/sql
    2、安装redis并启动服务
    3、maven打包运行或者IDE运行
